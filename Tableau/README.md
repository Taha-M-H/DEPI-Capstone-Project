# Tableau Phase — Visualization & Dashboards

File: `tableau final project fi.twbx`

Tableau was used purely for visualization, built on top of the dataset already cleaned and modeled in SQL/Python/Excel/Power BI — Tableau isn't intended for row-level data cleaning, so no cleaning logic lives in this phase.

## Dashboards

The workbook contains four dashboard pages:

1. **Demographics**
2. **Customers**
3. **Risk Analysis**
4. **Policy & Operations**
